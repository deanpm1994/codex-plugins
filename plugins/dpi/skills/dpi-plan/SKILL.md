# DPI Plan

Start the planning phase after the design spec is approved.

- Ask one clarifying question at a time if needed.
- If the answer is already in the repo, inspect it instead of guessing.
- Create one or more plan docs under `docs/dpi/plans/` when the work needs to be split into implementation slices.
- End each plan with a handoff section that records what the next phase should pick up.
- Keep the plan aligned with the approved spec; do not widen scope.
