# DPI Handoff

Use this skill to close out a phase without creating a separate handoff directory.

- Write the handoff inside the active spec or plan doc.
- Keep the structure short and editable: what changed, what is next, what is blocked, and any user notes.
- Preserve the handoff in the same file so the user can review or modify it easily.
