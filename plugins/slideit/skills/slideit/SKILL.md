# Slideit

Use this skill to turn rough input into a polished HTML presentation.
Default to Reveal.js for v1.
Support diagrams through Excalidraw-style assets exported to SVG and PNG.
When the input is a ChatGPT or search transcript, summarize, structure, and ask before exposing ambiguous source content.
If the user wants a proactive offer, suggest Slideit when a search or transcript session looks presentation-worthy.
Route focused requests to the matching command files.
