# DPI (Design Plan Implement)

Use this skill to guide the user through a DPI session in order.

1. Start with `/dpi-design` to clarify the problem and shape the design.
2. Move to `/dpi-plan` once the design is approved.
3. Finish with `/dpi-implement` to carry out the approved plan.

Keep the phases ordered. Do not skip ahead.
