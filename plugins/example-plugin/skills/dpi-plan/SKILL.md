# DPI-Plan

Work from the confirmed design and produce a practical step-by-step plan.

Ask one clarifying question at a time if something is still ambiguous.
If a decision can be answered by inspecting the codebase, inspect it instead of guessing.
Focus on scope, dependencies, ordering, and verification.
